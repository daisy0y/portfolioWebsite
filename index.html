About Me

포기하지 않는 개발자
유튜브의 알수없는 알고리즘으로 인하여 처음 접한 프로그래밍이 저의 새로운 꿈을 찾아주었습니다.
저는 잘하는 개발자는 아닙니다.
하지만 저의 목표는 프로게이머 시절부터 배워온 팀원 간의 협동, 성실함을 바탕으로
잘하면서 노력까지하는 개발자가 되는 것입니다.
게임이든 일이든 기본기가 탄탄해야 한다고 생각합니다. 항상 배움의 자세로 임하겠습니다.

PROJECT

WeaterToDo

시작페이지용 프로그램입니다. 페이지를 새로고침하면 배경 이미지가 변경이 됩니다. 로컬스토리지를 이용하여 TODOLIST와 이름 입력을 구현하였으며 Weater API를 이용하여 사용자 위치정보를 받아와 날씨정보를 제공하며 현재 시각을 실시간으로 보여줍니다.

  • Github

Tic Tac Toe

두명의 플레이어가 번갈아가며 3X3 보드판에 O와X를 번갈아 놓으며 3개의 칸이 연속되는 플레이어가 승리하는 게임입니다. 보드판은 Table으로 구성하였으며 승리,패배,무승부를 구현하였으며 카운터 기능을 추가하여 시간초과시 패배가 되도록 만들었습니다.
  • DEMO
  • Github

MEGABOX Website

기본적인 웹사이트의 구조를 이해하기 위하여 기업 사이트를 제작하였습니다. 미디어쿼리를 이용하여 반응형 웹사이트를 구현하였습니다. 유튜브 API를 연동하여 동영상을 삽입하였고 swiper api를 이용하여 이미지 슬라이드를 구현하였습니다.
  • DEMO
  • Github

Hit the Words

간단한 타자연습 게임입니다. 단어API를 이용하여 단어를 수집하고 무작위 단어가 추출되어 문제로 출제됩니다. 로컬스토리지 기능을 이용하여 점수를 저장 할 수 있게 하였으며 기록 확인란을 구성하였습니다.
